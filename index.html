This version of Midi converter is made for basic Midi files. This can make a simple Numbered musical notation (for Dizi/Xiao) from only one Track. Minimum note length can be 1/16, it allows single dotted note, and no grace notes, no bends, no vibrato, no tempo changing, etc.
